# FraudWatch

FraudWatch is a Java 21 / Spring Boot microservice platform for real-time fraud detection in banking flows.

## Current bootstrap status

- Root Maven monorepo initialized
- Shared library modules created
- Spring Boot service skeletons being bootstrapped

## Modules

- `libs/common-events`
- `libs/common-security`
- `libs/common-observability`
- `libs/common-test`
- `services/api-gateway`
- `services/auth-service`
- `services/transaction-service`
- `services/fraud-service`
- `services/review-service`
- `services/audit-service`
- `services/notification-service`
